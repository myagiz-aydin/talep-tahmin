# Talep Tahmin & Sipariş Optimizasyon Aracı

KOBİ'ler için yapay zeka destekli talep tahmini ve sipariş optimizasyon uygulaması.

## Ne Yapar?
- Geçmiş satış verilerini (CSV/Excel) analiz eder
- Facebook Prophet ile önümüzdeki dönemi tahmin eder
- Mevsimsellik ve trend analizi yapar
- Optimum sipariş miktarını hesaplar

## Teknolojiler
- Python
- Facebook Prophet
- Streamlit
- Pandas

## Nasıl Çalıştırılır?
```bash
pip install streamlit prophet pandas matplotlib
streamlit run uygulama.py
```
